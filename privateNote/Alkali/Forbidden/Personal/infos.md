Fact) ¬p ∧ p ↔ ⊥
Fact) ¬(±1 = +1) ↔ (±1 = -1)
p.f. 결론을 부정하여, p = (±1 = +1)에서, q = (±1 = -1) ≠ p로 놓자.
그러면, p ∧ q ↔⊥이므로, p와 q는 모순관계에 있으므로, 결론을 부정하면 거짓임을 알 수 있다.

Fact) (¬p → q), (p → q) ⊨ q
Fact) ((±1 = +1) → q, (±1 = -1) → q) ⊨ q
