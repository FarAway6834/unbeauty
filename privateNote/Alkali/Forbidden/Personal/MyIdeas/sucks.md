# 배고픈 사람의 부사 탐구

개인적 언어 (perosnal language): 개인만 쓰는 언어
비(非)부사-화(化) 정의 : 비(非)-부사인 품사형태 x와 부사형태 y에 대해, x의 사전적 뜻을 y를 이용하여 서술한 문장 s = "x는 f(y)"및, y를 이용하지 않고 사전적 정의를 서술한 문장 p에 대해, 문장 r = "x의 정의가 p일때, 문장 s를 만족하는 단어 y"라 하면, 부사 y의 뜻을 사전적으로 정의할때, {y : r}로 하는것.
평가용이부사(eval-ease adverb) : 부사를 비(非)-부사인 품사로 casting가능하기에, 비(非)부사-화(化) 정의가 가능한 부사.
악한 부사 (bad adverb) : 화자의 심리상태에 의존하고, 일반적인 화자가 사용하는 부사.
친절한 부사 (kind adverb) : 악한 부사가 아닌 평가용이부사
사악한 부사 (evil adverb) : 악한 부사이면서, 평가용이부사가 아닌 부사.
짖꿎은 부사 (naught adverb) : 평가 용이부사인, 악한 부사
사적 부사 : 개인적 언어에서 사용되는 부사.
공적 부사 : 공적 언어에서 사용되는 부사.
평가 용이 공적 부사 : 평가 용이 부사인 공적 부사
악한 공적 부사 : 악한 부사인 공적 부사
짖꾿은 공적 부사 : 짖꾿은 부사인 공적 부사
사악한 공적 부사 : 사악한 부사인 공적 부사
친절한 공적 부사 : 친절한 부사인 공적 부사
평가 용이 사적 부사 : 평가 용이 부사인 사적 부사
악한 사적 부사 : 악한 부사인 사적 부사
짖꾿은 사적 부사 : 짖꾿은 부사인 사적 부사
사악한 사적 부사 : 사악한 부사인 사적 부사
친절한 사적 부사 : 친절한 부사인 사적 부사
사적 언어 (private language) : 어휘 중에, 개인의 심리상태에만 의존하는 어휘가 존재하는 개인적 언어
이상한 언어 (weired language) : 개인적 언어인데, 사적 언어가 아닌 언어.
비형식적 이상한 언어 (informal weired language) : 형식언어가 아닌 이상한 언어

없을것같은 언어 (probably nonexistent language) : "어떤 자연어로도 각 어휘의 의미를 정의할수 없는" 비형식적 이상한 언어

골때리는 언어 (language but mind-boggling) : 다음을 만족
 - 형식언어가 아님.
 - 객관적인 문법이 존재함. (에초에 언어일 조건을 객관적인 문법이 존재할것으로 한정했음. 언어의 문장은 문법을 통해 동치류로 표현된 L*의 원소들의 집합이라 정의됨.)
 - 사적어격이 존재하지 않음.
 - 그러나 번역 불가능성이 존재함.
 
골때리는 공적 언어 : 골때리는 언어인데 공적.
골때리는 개인적 언어 : 골때리는 언어인데 이상한 언어. (사적 어격이 존재하지 않는 없을것같은 언어 (probably nonexistent language)의 부분집합. (진부분집합이라 한적 없다.))

참고 : 사적 어격의 존재 유무가 곧 사적 언어인지 유무라면, "골때리는 개인적 언어 = 없을것같은 언어"이다.

골때리는게 없을것같은놈일 조건 : 사적 어격의 존재 유무가 곧 사적 언어인지 유무. 이게 참이면, "골때리는 개인적 언어 = 없을것같은 언어"이다. ("사적언어"라는 단어의 표준 정의에선 사적 어격의 존재 유무가, 사적 언어인지 유무와 동치이므로, 이 조건은 참이다. (정방향함의는 당연히 참이고, 비트겐슈타인의 표준적 정의에 따르면, 비트겐슈타인이 《철학적 탐구》에서 "오직 나만 알아들을 수 있는 언어"의 가능성을 검토할 때, 그가 상정한 유일한 원동력이 바로 '사적 감각(Private Sensation)'이었으므로, 역방향 함의가 참))

진짜 없는가 문제 (is probably the same as definitely problem): "없을것같은 언어 (probably nonexistent language)는 진짜 없는가?"는 학문적 문제. (물론 증명할수 있다고 말한적 없다.)

배고픈 사람의 첫번째 정리 : <부사들이 평가 용이 부사들로만 이루어졌더라도, 그 부사들이 짖꾿은 부사이거나, 비부사-화 정의에 쓰이는 비-부사가 사적 어격을 가진다면, 해당 부사를 가지는 언어는, 공적으로 사용되지 않는 경우>, 사적 언어다.
배고픈 사람의 첫번째 추측 : 부사들이 평가 용이 부사들로만 이루어졌더라도, 그 부사들이 짖꾿은 부사이거나, 비부사-화 정의에 쓰이는 비-부사가 사적 어격을 가진다면, 해당 부사를 가지지지 않은 언어는, 공적으로 사용되지 않는 경우가 아닌 경우 사적 언어가 아닌 경우가 존재한다. (a.k.a. 진짜 없는가 문제는 거짓이라는 추측.)

현재 너무 배고파서, 글의 퀄이 떨어지는 중이기에, 서술을 마친다.
