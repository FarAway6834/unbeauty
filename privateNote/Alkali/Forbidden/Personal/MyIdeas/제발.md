# 미적분학이 직관을 말아먹은 학문이 아니고서야, 적분이 하나임.

D = d/dx라 치자,
Fact) x ∈ dom D⁻¹임은 기본적으로, x의 원시함수가 미분 가능하다는거임.

i) D⁻¹가 적분이라면
적분 가능성 == 원시함수의 미분 가능성

직관적인 시나리오지.

ii) D⁻¹는 적분에서 정의역을 제한한거지, 적분이 아닌거라면
적분은, 미분의 역과정을 포괄하는 더 상위의 연산일것임.

비직관적인 시나리오지.

다시 돌아와서,

th. 두 시나리오에 대해, 다음 결론이 나온다.
1. 문장 "적분 가능성 == 원시함수의 미분 가능성"과 동치인 모델에선, D⁻¹가 적분임.
2. 문장 "적분 가능성 ⊇ 원시함수의 미분 가능성"과 동치인 모델에선, D⁻¹이 적분이 아니고, 적분에서 정의역을 원시함수의 미분 가능성이 참인 함수들의 집합으로 줄여야 함.

결론 : 미적분학이, 직관을 말아먹은 학문이 아니고서야, f ∈ {df/dx | D¹ f}가 적분 가능한 f를 이르는 말이여야 함. 안그러면, "적분 가능성 ⊇ 원시함수의 미분 가능성"과 동치인 모델은 한두개가 아님. ㅈㄴ 여러개일거임
