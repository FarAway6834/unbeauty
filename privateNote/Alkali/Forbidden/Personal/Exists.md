# 존재 양화사 해설 : 콰인에 존재론에 대한 기호논리적 탐구 (취미생활)

나는 어재 (25.12.10) 까지도 존재 양화사가 뭔지 설명하기 어려웠었는데, python의 all과 any에 영감을 받아, 형식언어의 단어(문법사인 양화사) `∃`가 일상언어와 달리, 형식언어상의 존재론에서 어떻게 설명해야할지 햇갈렸다.

이제 다른사람이 이해하기 좋게, 해석 과정을 명확히 하고, 이에 대해 설명하겠다.

**먼저 무한 논리곱의 관점으로 시작한다**

1. ∧{x, y} = x ∧ y
2. ∧(S∪{F}) = F
3. ∧(S∪{T}) = ∧S

라 하면,

Fact) {T, T} = {T} 으로, ∧{T} = ∧∅ = T

1. ∧∅ = T
2. ∧(S∪{F}) = F
3. ∧(S∪{T}) = ∧S

라 하면,

1. ∧{x, F} = F = x ∧ F
2. ∧{x, T} = ∧{x}인데, ∧{T} = ∧∅ = T = T ∧ T 이고, ∧{F, T} = ∧{T, F} =(1번)= F
3. ∧{x, y} = x ∧ y

이므로,

∧{x, y} = x ∧ y이든 ∧∅ = T이든 같은 규칙이 됨. 1번규칙은 2, 3번규칙에서 서로 논리적 동치임이 만족되고,

3번규칙 ∧(S∪{T}) = ∧S는 2번규칙과 ∧{x, T} = x이 동등함을 만족한다.

주사와 빈사를 통한 양화문장은,

∀x∈S, V(x) : ∧_{x∈S} V(x)
∃x∈S, V(x) : ∨_{x∈S} V(x)

에서,

∨_{x∉∅} V(x) : ¬∧_{x∉∅} ¬V(x)
∧_{x∉∅} V(x) : ¬∨_{x∉∅} ¬V(x)

로, 

∀x∉∅ V(x) : ¬∃x∉∅, ¬V(x)
∃x∉∅ V(x) : ¬∀x∉∅, ¬V(x)

이다.

참고로, (전체집합은 공집합의 여집합) U = ∅^∁ 으로, 주사가 U였다면, 보편적인 양화문이 된다.

위에서 밝힌 존재기호의 특성에 따라,

∄x Φ(x) : ∀x ¬Φ(x)

이다.

따라서, 콰인의 존재론 (EXSTAT ALIQUIS의 EXSTAT)의 존재 개념 (EXSTAT) `∃`는

x ⊨ Φ (술어 Φ를 변항 x가 만족한다) : Φ(x) 인데, 모델론적으로 술어라는 진리함수는

집합의 맴버십 (∈) 함자이므로,

Φ(x) : x ∈ Φ

이기까지 하다.

(∃)x : ((x ⊨ Φ) → Φ ≠ ∅)

이것이 콰인의 존재론이 되는데, 이 관점에서,

(∄x, Φ(x)) : (⊭ Φ) : (∀x ¬Φ(x)) : (Φ = ∅)
(∃x, Φ(x)) : (¬(⊭ Φ)) : (¬∀x ¬Φ(x)) : (Φ ≠ ∅)
(∀x, Φ(x)) : (⊨ Φ) : (∄x, ¬Φ(x)) : (Φ = U = ∅^∁)
(생략)

그렇자. 전칭양화란, 모든것의 양만큼 양화된것이다. 모든것의 양만큼 양화된 x에 대해 Φ(x)이라면, Φ(x)이지 않는 경우는 없다.

(생략된걸 마저 이어서)
(∃!x, Φ(x)) : (Φ = {x})

유일성은 유일한 주사에 대한 양이 된다.

term logic에서의 양화는 이러한 속성을 가지게 된다.

그리고, 이 언어게임 (비트겐슈타인적 관점)의 생성문법 (노엄 촘스키의 관점)의 문법규칙 (형식언어는 속뜻없이 고정된 해석이 적용된다는 뜻으로 약속되었다는 의미론의 관점)은

(∀x, Φ(x)) : Φ

으로, Φ^∁ = ∅가 된다.

그렇다면 이것은 무엇인가?

새로운 해석방법인 **진리 함자 의미론**을 만들어보겠다.

`s/([PredicatalSymbol])\(([VariableSymbol])\)\b[\s\t]*\b:\b[\s\t]*\b(.*?)/\(∈$1\)\($2\)="$3"/`인 모델론적 규칙을 따른다.

언어 L에 대해, t ∈ L* 일때,

"Φ(x) : t" : (∈Φ)(x) = "t" 인 함자적 특성을 가진다.

그것이 무슨 의미인가?

빈칸 □에 대해, 매개변수 x와 진리함수 f(x) = □에 대하여,

`/^f\((.*)\)$/`로 감지한 `$x = $1`에 대해,

`s/x/${$x}/`로

빈칸에 들어갈 값으로 일괄 치환해서 평가한다는거다.

이렇게 이론전산•형식언어적 의미론으로 해석된다.

음... 근데 대체 고정이 뭔데?

아래로 설명된다.

fact) "고정된 절차는 튜링 기계로 해석될수 있다."

형식언어는 고정성을 가지고, 술어의 의미는 진리함수적으로 평가될때, 비로소 이와 관계되어 가지는 양화사들의 의미가 생기게 된다.

즉, 콰인의 존재론 (EXSTAT ALIQUIS의 EXSTAT)의 존재 개념 (EXSTAT) `∃`은 ((당연하지만) 형식언어 내에서) 바인딩하는 (=존재론적으로 언질하는) 문장이 무한 논리곱에 의해서 평가될때, 그 의미가, "Φ(□)"에서 빈칸에 알맞은 값을 채워넣을수 있는 Φ에 대한 특칭문장을 나타내어, ~~(1형식이라서 빈사가 없지만)~~ 주사의 존재성이, 주사인 대상을 댈수 있게 하는 형식언어적인 뜻에 한정하게 된다.

즉, 형식 논리 언어는 일상 논리 언어의 일부분만 담고 있는 번역 불확정성을 가진다는 말이 맞다.

이런식으로 증명해서 직관을 길르니까, 형식언어를 하는 재미가 있는것 같다.
